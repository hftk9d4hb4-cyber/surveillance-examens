# Notes de version — V1.1.0-rc.1

Cette version est une **Release Candidate** destinée à la recette avant la rentrée universitaire.

## Objectif

Stabiliser le cœur fonctionnel sans ajouter de modules non indispensables : imports, disponibilités, affectations, convocations, droits et documentation d’exploitation.

## Points à surveiller pendant la recette

- délivrabilité des e-mails Gmail ;
- compatibilité des fichiers ICS avec Outlook et Google Agenda ;
- qualité des fichiers réels d’import ;
- temps de calcul sur une campagne complète ;
- compréhension des messages par les gestionnaires ;
- comportement lors de la correction d’un examen déjà affecté.

## Limites acceptées pour la V1.1

- pas de SSO institutionnel ;
- pas de synchronisation ADE/Apogée ;
- pas d’annulation automatique des invitations calendrier ;
- pas de limitation de connexion distribuée ;
- pas de sauvegarde applicative intégrée ;
- pas d’application mobile.

Ces points ne bloquent pas une première utilisation encadrée dans une faculté unique.
