# Recette Sprint 2
1. Un enseignant ajoute puis supprime une absence.
2. Une période dont la fin précède le début est refusée.
3. Les préférences sont enregistrées par jour et demi-journée.
4. Un enseignant ne peut agir sur les données d'un autre enseignant.
5. Un gestionnaire accède au tableau consolidé.
6. Un enseignant ne peut pas accéder au tableau consolidé.
7. Les données V1.1 restent présentes après migration.
